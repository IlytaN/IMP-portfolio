Getting familiar with jQuery

1. jQuery is a javascript library. It helps design the client side scripting of HTML. We can manipulate DOM easier, create animation, handle events and develop ajax application.

2. It helps web developers work with javascript easily, create abstraction for low-level interaction and animation, advanced effects and high-level, themeable widgets. The modular approach to the jQuery library allows the creation of powerful dynamic web pages and Web applications.

3. I can create animations for that app. I can also do DOM easily.

4. CDN is Content Delivery Network. MaxCDN, Google and Microsoft host it. With it, we only need to include a link in the header; we don't need to have the whole jquery library in our files
