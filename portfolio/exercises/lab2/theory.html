RWD

a.What is the purpose of the RWD? Why it is used? How things are done before RWD? What is the difference?

The purpose of RWD is to make the design flexible, which means that the design can beautifully adapt to any devices.

It is used because we cannot just make a specific design for every device. The technology trends change so fast that we cannot do that way all the time.

Things are done with proponent of non-fixed layouts for a long time. Then people come with the idea of grid system, the design is fixed and fluid.

The difference is that instead of tailoring disconnected designs to each of all web devices, We can design for an optimal viewing experience, but embed standards-based technologies into our designs to make them not only more flexible, but more adaptive to the media that renders them

b.How css media queries can help you building responsive web sites?

A media query allows us to target not only certain device classes, but to actually inspect the physical characteristics of the device rendering our work.

c.Why to use em (font) based sizing for example margins?

em: Relative to the font-size of the element. Using em for sizing margin is good because it helps the layout more apparent to the users.
