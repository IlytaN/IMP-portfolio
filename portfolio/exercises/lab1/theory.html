Lab 1

Definition

a.HTML is Hyper Text Markup Language. It's like a frame of a website. It describes the content of the website.

b.CSS is Cascading Style Sheets. It's used to create the appearance of the website.

c.Javascript is a programming language. It can be used to make the website interactive.

d.HTTP is HyperText Transfer Protocol. It is used to define the format and transmit method of the message and it decides the actions for the web server and browser.

e.HTML5 is a version of HTML. Currently it is the most popular version.

f.Dynamic HTML includes some technologies that make animated and interactive website. DHTML often includes a HTML file, a CSS file and a javascript file.

Coding convention

It's a guideline for how to write codes (structure, statements etc.) so that it's easy to be understood and edited. Following the convention is important because that makes all the coders understand and don't waste time trying to learn different style. I like to use: with brace placement, I often use K&R style
