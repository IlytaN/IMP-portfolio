What I have learnt

I have learnt how to make responsive web design, how to use css media queries, "em" etc.

I have also learnt what is Web socket and how to use it. Local storage is also a new thing to me.

Plus, I have learnt how to work with canvas in Javascript

I also had the chance to try jQuery and ajax. I had tried some other javascript libraries before , this time I didn't see it as a new thing, but its syntax is different, and that caused some troubles for me

In lab 6, I got a lot of troubles with jsonp and jquery, but finally I made it done

The teacher's advise was also useful for me. I didn't notice before that splitting the exercise into smaller tasks and checking error by console.log were so powerful.

Responsive web design is not always easy, but I think it's still easier than dealing with server and data

Difficulties

In jQuery parts, I got many problems because the syntax was different. When it came to ajax and jsonp, things were even more difficult
